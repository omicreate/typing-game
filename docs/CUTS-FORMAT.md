# カット（セリフ）追加ガイド

このゲームの物語は `index.html` 内の `CUTS` 配列で定義します。1要素＝1カット（1セリフ）です。ビルド不要・1ファイル完結なので、ここを編集して保存するだけで物語が変わります。

## スキーマ

```js
cut(who, theme, jp, kana)
```

| 引数 | 意味 | 例 |
|---|---|---|
| `who` | 話者名（画面のチップに表示） | `"ひなた"` |
| `theme` | 配色・アバター文字のプリセット（下記 `T.*`） | `T.hinata` |
| `jp` | 画面に表示する日本語セリフ。**記号OK**（、。…！？） | `"ねえ、星が見える日かな。"` |
| `kana` | **打鍵対象の読み（ひらがな）**。記号・スペースは入れない | `"ねえほしがみえるひかな"` |

`jp` は「見せる文」、`kana` は「打つ読み」。役割が違うので別々に書きます。

## 配色プリセット `T.*`

`CUTS` の上で定義済み。新キャラを足したいときはここに1行追加します。

```js
const T = {
  nare:   { glyph: "★", c1: "#8a7dff", c2: "#3b2f6e", g1: "#241c47", g2: "#120f28" }, // ナレ
  hinata: { glyph: "ひ", c1: "#6d5efc", c2: "#9b8bff", g1: "#2b2150", g2: "#141026" }, // 主人公（藍）
  sora:   { glyph: "そ", c1: "#ff7a59", c2: "#ffb199", g1: "#3a2238", g2: "#161021" }, // 相手（橙）
  futari: { glyph: "☆", c1: "#6d5efc", c2: "#ff7a59", g1: "#2b2150", g2: "#161021" }, // 二人同時
};
```

- `glyph`: アバター円の中の1文字（絵文字も可）
- `c1`/`c2`: アバターのグラデーション色
- `g1`/`g2`: 背景グラデーション（上/下）

## kana の書き方ルール（重要）

入力エンジンは **ひらがなの読み**を解釈します。次を守れば崩れません。

- **ひらがなのみ**。カタカナは使わない（読みをひらがなに直す）。
- **長音符 `ー` は使わない** → 母音で書く（例: せーの → `せえの`、きょー → `きょう`）。
- **記号・スペースは入れない**（`、。…！？` は `jp` 側だけ）。
- **語末に小さい「っ」を置かない**（例: 「お願いっ」→ `おねがい` にする）。
- 促音「っ」は**必ず次の音がある**こと（例: `って`＝っ+て）。

## ローマ字入力エンジンの仕様（参考）

打ち方は1通りに固定せず、**複数のローマ字を許容**します（`し=shi/si/ci`、`つ=tsu/tu`、`ち=chi/ti`、`ふ=fu/hu`、`じ=ji/zi`、`を=wo/o`、`しゃ=sha/sya` など）。特殊音は次の通り。

- **促音「っ」**: 次の子音を重ねる（`って=tte`、`っし=sshi/ssi`）。または `xtu`/`ltu`。
- **撥音「ん」**: 常に `nn` でOK。次が**子音（や行以外）**なら単独 `n` も可。次が**母音・や行・語末**は `nn` 必須（`こんなに=konnani`、`こんやく=konnyaku`）。

## カットを追加する手順

1. `index.html` の `CUTS` 配列に `cut(...)` を1行足す。
2. 話者を増やすなら `T` にプリセットを追加して使う。
3. 保存してブラウザを再読み込み（カット数・進行ドットは自動でカウントされます）。

## 追加後の自己チェック（任意）

読みに打てない文字が混じっていないかは、ブラウザのコンソールで確認できます。

```js
// ページを開いた状態で実行：全カットの読みが打ち切れるか検査
const { toUnits, evalState, cands } = window._engine;
[...document.querySelectorAll('script')]; // ※ kana は CUTS で定義
// 例）1つの読みを検査
const k = "せえのおねがい";
const u = toUnits(k);
evalState(u, u.map((_,i)=>cands(u,i)[0]).join("")).complete; // true なら打てる
```

`true` が返れば、その読みは（少なくとも1通りで）打ち切れます。
