# 星待ちの放課後 - セリフタイピング

表示されたセリフをローマ字で打つと、放課後の小さな物語のカットが進む、PC向けのセリフタイピングゲームです。
単一HTMLで動作し、ビルド不要で遊べます。

ローマ字は複数の打ち方に対応しています。`し=shi/si`、`つ=tsu/tu`、`ち=chi/ti`、`ふ=fu/hu`、`じ=ji/zi`、`を=wo/o`、`ん=n/nn`、`っ=子音重ね/xtu` などを、かなベースの入力エンジンで判定します。

## 公開URL

- 遊ぶ: https://omicreate.github.io/hoshimachi-serifu-typing/
- リポジトリ: https://github.com/omicreate/hoshimachi-serifu-typing

## 主な機能

- オリジナル脚本の全25カットを打ち切るタイピングゲーム
- 複数スペルを許容するローマ字入力
- 打鍵ごとの着色、コンボ、正確率、タイム、ランク表示
- 外部ライブラリ、外部API、ビルド工程なし

## 使い方

1. 公開URLをPCのブラウザで開きます。
2. 表示されたローマ字をキーボードで入力します。
3. 全カットを打ち切ると、結果ランクが表示されます。

セリフを差し替える場合は、`index.html` 内の `CUTS` 配列を編集します。
書き方の詳しい仕様は [docs/CUTS-FORMAT.md](docs/CUTS-FORMAT.md) を確認してください。

```js
cut("ひなた", T.hinata, "日本語のセリフ", "うつよみがな"),
```

`kana` はひらがなの読みだけを書きます。記号、長音符、語末の小さい「っ」などのルールは [docs/CUTS-FORMAT.md](docs/CUTS-FORMAT.md) にまとめています。

## データ/プライバシー

- アカウント登録は不要です。
- 外部APIへの送信はありません。
