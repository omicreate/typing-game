# 星待ちの放課後 — セリフタイピング

表示されたセリフを**ローマ字**で打つと、放課後の小さな物語のカットが進む、**縦型（9:16）のセリフタイピングゲーム**。単一HTMLで動作し、ビルド不要。

## 遊ぶ

👉 **https://omicreate.github.io/typing-game/**

- 表示されたローマ字をそのまま入力（スペースは自動でスキップ）
- 全9カットを打ち切るとランク（S/A/B/C）が出る
- キーボード推奨。スマホは画面タップで入力欄が出ます

## 特徴

- オリジナル脚本（著作物の流用なし）
- 縦型レイアウトでスマホ・ショート動画映え
- ローマ字入力エンジン（打鍵ごとの着色・コンボ・正確率・タイム）
- 外部ライブラリ・APIなし。`index.html` 1ファイルで完結

## ローカルで動かす

```bash
# どれか1つ
npx serve typing-game     # → http://localhost:3000
python3 -m http.server    # typing-game/ で実行
```
または `index.html` をブラウザで直接開くだけ。

## カスタマイズ

`index.html` 内の `CUTS` 配列を編集すると、セリフ・話者・配色を差し替えられます。

```js
{ who: "話者名", glyph: "顔文字/文字", c1: "#色1", c2: "#色2",
  g1: "#背景上", g2: "#背景下",
  jp: "日本語のセリフ", romaji: "nyuuryoku suru romaji" }
```
