# 星待ちの放課後 — セリフタイピング

表示されたセリフを**ローマ字**で打つと、放課後の小さな物語のカットが進む、**横型のセリフタイピングゲーム**（PC向け）。単一HTMLで動作し、ビルド不要。

ローマ字は **複数の打ち方に対応**（`し=shi/si`、`つ=tsu/tu`、`ち=chi/ti`、`ふ=fu/hu`、`じ=ji/zi`、`を=wo/o`、`ん=n/nn`、`っ=子音重ね/xtu` など）。かなベースの入力エンジンで判定します。

## 遊ぶ

👉 **https://omicreate.github.io/typing-game/**

- 表示されたローマ字を入力（打ち方は複数OK）
- オリジナル脚本・全25カットを打ち切るとランク（S/A/B/C）が出る
- パソコンのキーボードで遊ぶゲームです

## 特徴

- オリジナル脚本（著作物の流用なし）
- 横型レイアウト（PCで快適に読める）
- かなベースのローマ字入力エンジン（複数スペル許容・打鍵ごとの着色・コンボ・正確率・タイム）
- 外部ライブラリ・APIなし。`index.html` 1ファイルで完結

## ローカルで動かす

```bash
# どれか1つ
npx serve typing-game     # → http://localhost:3000
python3 -m http.server    # typing-game/ で実行
```
または `index.html` をブラウザで直接開くだけ。

## カスタマイズ・セリフ追加

`index.html` 内の `CUTS` 配列を編集すると、セリフ・話者・配色を差し替えられます。書き方の詳しい仕様（kanaの書き方ルール、配色プリセット、入力エンジンの挙動）は **[docs/CUTS-FORMAT.md](docs/CUTS-FORMAT.md)** を参照。

```js
// CUTS 配列に1行足すだけ。who=話者, T.*=配色プリセット, jp=表示文, kana=打つ読み
cut("ひなた", T.hinata, "日本語のセリフ（記号OK）", "うつよみがな"),
```

`kana` はひらがなの読みだけ（記号・長音符 `ー`・語末の小さい「っ」は不可）。詳細は [docs/CUTS-FORMAT.md](docs/CUTS-FORMAT.md)。
